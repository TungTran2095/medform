# **App Name**: PlanForm 2026

## Core Features:

- Unit Information Input: Collect basic information about the unit, including name, type, leader, and the person responsible for planning.
- SWOT Analysis Input: Provide dedicated sections for inputting Strengths, Weaknesses, Opportunities, and Threats.
- 2026 Priority Ranking AI Tool: Suggest a prioritized list of 5-7 key initiatives for 2026, based on the SWOT analysis and a reasoning tool analyzing existing data. Requires admin confirmation of proposal before application.
- KPI Suggestion AI Tool: Automatically suggest relevant KPIs based on the inputted objectives, with a reasoning tool evaluating suitability.
- Form Submission: Allow users to submit the completed form to Firestore.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) for a calm and professional feel.
- Background color: Light gray (#F0F0F0) for a clean and neutral canvas.
- Accent color: Muted purple (#BDB2FF) to highlight key areas without being overwhelming.
- Body font: 'Inter', a sans-serif for a modern, clean, and highly readable style.
- Headline font: 'PT Sans', a sans-serif used for titles and short sentences.
- Clean, minimalist layout with clear section headings to guide the user through the form.
